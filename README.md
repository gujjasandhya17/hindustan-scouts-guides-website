# Hindustan Scouts & Guides Association (HSGA) Website

## Project Overview

This project is a **responsive multi-page website** developed for the **Hindustan Scouts & Guides Association (HSGA)** using **HTML, CSS, and JavaScript**.

The website provides information about the organization including its **mission, leadership, activities, gallery, and contact details**. It is designed to be simple, responsive, and easy to navigate across different devices.

---

## Live Website

The project is deployed online and can be accessed here:

🔗 https://hindustan-scouts-guides-website.netlify.app/

---

## Tech Stack

- HTML5
- CSS3
- JavaScript 


## Project Features

- Responsive design (Mobile, Tablet, Desktop)
- Sticky navigation bar
- Mobile hamburger menu
- Image gallery with lightbox preview
- Contact form with client-side validation
- Hover effects and smooth user experience

---

## Website Pages

**Home (index.html)**
- Introduction to the organization
- Vision and mission
- Quick links and activity highlights

**About (about.html)**
- History and background of HSGA
- Mission and vision details

**Leadership (leadership.html)**
- National leadership
- Organizational structure

**Activities (activities.html)**
- Training programs
- Community service initiatives
- Youth development activities

**Gallery (gallery.html)**
- Image grid displaying activities
- Lightbox image preview

**Contact (contact.html)**
- Organization contact details
- Contact form for inquiries

## Deployment

This project is deployed using **Netlify**.

Steps used for deployment:

1. Push the project to GitHub.
2. Connect the GitHub repository to Netlify.
3. Deploy as a **static site**.
4. No build command required.

Live URL:
https://hindustan-scouts-guides-website.netlify.app/
